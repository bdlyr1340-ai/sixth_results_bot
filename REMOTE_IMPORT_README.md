# الاستيراد المباشر من مركز نتائجنا

هذه الإضافة تلغي الحاجة إلى رفع ملف ZIP من الهاتف.

## طريقة الاستخدام

1. افتح `https://center.mlazemna.com/`.
2. ادخل إلى مجلد المديرية/المحافظة ثم اختر `علمي` أو `أدبي`.
3. انسخ رابط المجلد من شريط المتصفح، وليس رابط الصفحة الرئيسية.
4. افتح لوحة مالك البوت.
5. من بطاقة المحافظة افتح **جلب مباشر من مركز نتائجنا**.
6. ألصق الرابط وحدد السنة والدور والفرع، ثم اضغط **جلب الملفات وبدء الفهرسة**.

في بغداد يمكن وضع عدة روابط، رابط الكرخ الأولى والثانية والثالثة والرصافة الأولى والثانية والثالثة، كل رابط في سطر مستقل داخل عملية واحدة.

Railway يكتشف ملفات PDF وينزل كل ملف بصورة مؤقتة ويقرأه ثم يحذفه مباشرة. لا تبقى ملفات PDF كاملة مخزنة بعد انتهاء العملية؛ تبقى قاعدة بيانات البحث فقط.

## ملفات التحديث

استبدل الملفات الموجودة في الحزمة مع الحفاظ على المسارات نفسها، وأضف الملف الجديد:

- `app/importer/remote.py` — ملف جديد
- `app/importer/jobs.py`
- `app/config.py`
- `app/web/server.py`
- `app/web/templates/base.html`
- `app/web/templates/dashboard.html`
- `app/web/templates/import_status.html`
- `.env.example`

لا تحتاج إلى تغيير قاعدة البيانات أو حذف البيانات القديمة. بعد رفع الملفات إلى GitHub، انتظر نشر Railway الجديد.
